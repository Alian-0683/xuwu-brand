Mission

使命

我们存在的意义，是帮助每一个空间找到它的表达——无论是正在成长的品牌、认真生活的家庭、还是被忽视的银发一族。我们相信好的设计不只是装饰，而是让功能性、美学与情感共存。

用设计叙事，让空间为每一位使用者说话。

Vision

愿景

成为成都值得信赖的复合型空间设计力量。

规模目标 — 3-5年内完成从工作室到公司的升级，团队能力与品牌影响力同步成长
专业目标 — 在成都设计圈建立可识别的口碑，作品说话，案例传播
团队目标 — 让每位设计师都拥有跨类型的设计能力和对好设计的判断力
社会目标 — 为认真做品牌的人、认真生活的人、有真实需求的人做好设计
Core Values

核心价值观

VALUE 01

叙事优先，不做容器

空间不是货架，是语言。

每一个项目都值得被认真对待。我们拒绝"放进去能用的设计"，追求"说清楚使用者是谁的设计"。用物件叙说故事，是手艺，也是底线。

  • 接项目前，先问：这个空间要讲什么故事？
  • 不做没有逻辑支撑的视觉堆砌
  • 用设计语言解决商业问题，而不是用装饰掩盖问题
VALUE 02

功能与体验，共生共存

好设计让使用者感受到被理解。

功能是底线，体验是追求。我们不追风格，追的是——让人愿意待、愿意买、愿意回来。动线、视觉、节奏，所有设计动作都指向同一个结果：真实的体验发生。

  • 设计方案第一步永远是梳理功能逻辑
  • 美学服务于功能，不做无效的"好看"
  • 用设计引导行为，而不是强迫适应
VALUE 03

真诚交付，不做妥协

信任是双向的，选择也是双向的。

我们用作品回报每一位客户的信任。对于方向不一致的合作，我们选择不接——这是对客户的负责，也是对团队时间的尊重。

  • 不接低预算、低审美、无合作基础的项目
  • 有话直说，不为了签单做虚假承诺
  • 交付的作品，至少让自己满意
VALUE 04

设计者有所为

做设计是一份手艺活，值得被认真对待。

我们希望团队成员：能被甲方认可，能做出让自己骄傲的作品，能在有价值的项目中获得成长。能传播是加分项，但前提是先让自己满意。

  • 每个项目都有设计师的个人署名和复盘
  • 鼓励团队参与能展示个人风格的项目
  • 认可"被行业传播"是职业成就的一部分
VALUE 05

空间无边界，设计有态度

好的设计师不被类型限制。

零售、家装、适老化——只要是空间，都值得被认真对待。我们培养的是能驾驭多种空间类型的复合型人才，类型不是边界，能力才是。

  • 鼓励设计师接触不同类型的项目
  • 团队内部共享跨类型的设计经验
  • 不设"只能做某类空间"的职业天花板
Spirit

企业精神

不追潮流，追答案。

不跟风，不贴标签，不做表面的风格堆砌。每一个项目的出发点永远是：

这个空间要解决什么问题？
它的主人是谁？
怎样才能让使用它的人感受到被理解？
Business

业务定位

空间是容器，也是语言。

叙物不设业务边界，专注以下领域：

业务类型 核心关注
零售空间 品牌叙事、动线逻辑、转化体验
家居空间 生活场景、功能优化、情感归属
适老空间 安全实用、尊严生活、真实需求
其他空间 只要有真实需求，都值得被认真对待

服务理念：功能导向，体验落地，品牌叙事，复合执行。

Talent

人才理念

在叙物，设计师不做专才，做全才。

我们希望你：

  • 01

    对好设计有标准，对烂设计有底线

  • 02

    能跟客户讲清楚"为什么这样设计"

  • 03

    在甲方认可之前，先让自己满意

  • 04

    愿意在不同类型的空间里积累经验和视野

我们提供：

真实项目的跨类型实践机会
鼓励个人风格的设计土壤
让作品说话的展示平台
Guidelines

团队行为准则

每个叙物人，在任何一个项目里都应遵循：

  • 01

    先理解，再设计

    理解空间的使用者，理解他们真实的需求

  • 02

    功能在前，美学在后

    解决实际问题，再谈风格表达

  • 03

    有话直说

    对客户、对队友、对自己的想法负责

  • 04

    交付满意的作品

    至少让自己觉得这件事做得值

  • 05

    持续成长

    不固守已有经验，保持对设计的敏感和好奇
